# Finance Tracker API

A REST API built with Node.js and Express for managing financial transactions.

## Features
- Add transactions
- Get transactions
- Update transactions
- Delete transactions
- Summary calculation (income, expense, balance)

## Tech Stack
- Node.js
- Express
- Postman

## Run the Project

Install dependencies

npm install

Start server

node server.js

Server runs on

http://localhost:5000

## API Endpoints

POST /api/transactions  
GET /api/transactions  
PUT /api/transactions/:id  
DELETE /api/transactions/:id  
GET /api/summary
